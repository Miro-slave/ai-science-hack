# AI Science Hack

Проект для анализа изображений образцов руды с использованием моделей
компьютерного зрения. Реализует полный пайплайн подготовки данных,
обучения моделей сегментации и классификации, а также инференс через
веб-интерфейс.

## Стек технологий

-   **Python 3.12**
-   **PyTorch**
-   **segmentation-models-pytorch**
-   **timm**
-   **Albumentations**
-   **OpenCV**
-   **NumPy**, **scikit-learn**, **Matplotlib**
-   **TensorBoard**
-   **Streamlit**
-   **Docker**

## Пайплайн

Проект включает две задачи:

-   **Сегментация** - поиск областей талька.
-   **Классификация** - определение типа руды.

Этапы обработки:

1.  Подготовка изображений.
2.  Генерация масок.
3.  Нарезка изображений на перекрывающиеся чанки.
4.  Аугментация и разделение данных на train/test.
5.  Обучение моделей с логированием в TensorBoard и сохранением лучших
    весов.
6.  Инференс:
    -   классификация изображения;
    -   сегментация;
    -   объединение результатов;
    -   построение heatmap, бинарной маски и итоговой визуализации.

Во время инференса изображение также разбивается на чанки, а результаты
объединяются с использованием гауссового взвешивания.

### Сборка и запуск

``` bash
docker build -t nornikel .
docker run -p 8501:8501 nornikel
```

После запуска веб-интерфейс Streamlit будет доступен по адресу:

``` text
http://localhost:8501
```
